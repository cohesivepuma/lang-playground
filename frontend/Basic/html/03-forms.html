HTML 表单

一、表单结构 <form>

action —— 表单提交的 URL 地址

method —— 提交方式：

  • get（默认）—— 数据附在 URL 后，有长度限制，可被浏览器缓存/收藏
  • post —— 数据在请求体中，无长度限制，更安全

二、Input 属性速查

属性 说明
type输入框类型（见下表）
name表单提交时的字段名（必须有）
id元素标识，供 label、CSS、JS 使用
value输入框的值（预填内容）
placeholder占位提示文本（灰色）
required必填项验证
maxlength最大输入长度
minlength最小输入长度

三、Input Type 类型速查

类型 说明
text文本输入（默认）
password密码输入（内容显示为圆点）
email电子邮件（自带 @ 格式验证）
number数字输入（带上下箭头）
tel电话号码（移动端弹出数字键盘）
urlURL 地址（自带格式验证）
date日期选择器
time时间选择器
range范围滑块
radio单选按钮（同 name 互斥）
checkbox多选框
file文件上传
submit提交按钮
reset重置按钮
button普通按钮（无默认行为）
