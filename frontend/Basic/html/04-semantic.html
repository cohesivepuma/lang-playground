HTML5 语义化入门

什么是语义化？

用有意义的标签描述内容结构，而不是只用 div 和 span。

四大好处

  1. 代码可读性 —— 结构清晰
  2. SEO 优化 —— 搜索引擎更好理解结构
  3. 可访问性 —— 屏幕阅读器正确解读
  4. 样式分离 —— 减少 class 依赖

语义化标签速查

标签说明
<header>页面或区块的头部
<nav>导航链接区域
<main>页面主要内容（唯一）
<article>独立的文章内容
<section>文档中的区块
<aside>侧边栏/附属内容
<footer>页面或区块的底部

<title> vs <h1>

标签位置作用
<title>head 中，不可见浏览器标签页 + 搜索结果标题 + 收藏书签名
<h1>body 中，可见页面内的大标题
