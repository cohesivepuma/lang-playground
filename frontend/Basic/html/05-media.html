媒体元素

一、图片 <img>

基本用法

img 属性速查

属性说明
src图片路径（必须）
alt替代文字，图片无法显示时展示
width / height图片宽高（建议写上防止布局抖动）
title鼠标悬浮显示的提示信息
srcset响应式图片，不同分辨率加载不同图片

响应式图片 (srcset)

浏览器会根据视口宽度自动选择合适尺寸的图片：

<img src="small.jpg"
     srcset="small.jpg 480w, medium.jpg 800w, large.jpg 1200w"
     sizes="(max-width: 600px) 480px, 800px"
     alt="响应式图片">

二、视频 <video>

video 属性速查

属性说明
src视频路径
controls显示播放/暂停/进度条/音量控制
width / height视频宽高
autoplay自动播放（需配合 muted）
muted静音（autoplay 必须配合 muted）
loop循环播放
poster封面图片（视频加载前显示）
preload预加载：auto / metadata / none

多源视频

<video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    您的浏览器不支持视频播放
</video>

视频字幕 (track)

<video controls>
    <source src="video.mp4" type="video/mp4">
    <track src="subtitles.vtt" kind="subtitles" srclang="zh" label="中文">
</video>

三、音频 <audio>

audio 属性速查

属性说明
src音频路径
controls显示控制条（播放/暂停/进度/音量）
autoplay自动播放
muted静音
loop循环播放

多源音频

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    您的浏览器不支持音频播放
</audio>

五、SVG 矢量图

SVG 是矢量图格式，放大不失真，适合图标和 Logo。

fill="currentColor" 让图标颜色跟随父元素文字颜色，暗色模式自动适配。

SVG path 指令

指令含义
MMove to（移动到起点）
LLine to（画直线）
CCurve to（贝塞尔曲线）
ZClose（闭合路径）

SVG 示例

circle

rect

path (heart)
