# HTML 基础教程

> 整理自 Obsidian 笔记 `notes/90-programming/frontend/html/`，转为可直接运行的 HTML 文件。

## 文件列表

| 文件 | 内容 |
|------|------|
| `01-basic-structure.html` | 文档结构、标题、段落、块/行内元素、路径、注释 |
| `02-text-and-links.html` | 文本格式化（加粗/斜体/高亮/上下标）、超链接、span、三种列表 |
| `03-forms.html` | 表单结构、所有 input type、select、textarea、表单验证 |
| `04-semantic.html` | HTML5 语义化标签、SEO 元标签、figure/figcaption、实体字符 |
| `05-media.html` | 图片(响应式)、视频(多源/字幕)、音频、表格(合并)、SVG |

## 使用方式

直接用浏览器打开任意 `.html` 文件即可查看效果。

```bash
# 例如用默认浏览器打开第一课
open 01-basic-structure.html
```
